# 大语言模型

## Definition

Large Language Model (LLM) — 参数量在数十亿到数万亿之间的语言模型，基于[[Transformer]]架构，通过海量文本预训练获得广泛的语言理解和生成能力。

## Key Aspects

- 核心架构：[[Transformer]]（Decoder-only为主）
- 训练三阶段：预训练 → SFT → [[RLHF]]
- 关键规律：[[Scaling Law]]
- 代表模型：[[GPT系列]]、[[LLaMA]]、[[Mistral]]
